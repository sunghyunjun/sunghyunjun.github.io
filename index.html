Shared Library Path 이해와 cgo 사용하기

Shared Library를 cgo와 함께 사용하는데 있어 도움이 되는 정보를 기록합니다. LD_LIBRARY_PATH LD_LIBRARY_PATH는 각종 library를 사용할 때 참조 path 문제를 해결하고자 사용하곤 합니다. export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/path/to/shared/lib 위와 같은 식으로 사용이 가능합니다. 하지만 LD_LIBRARY_PATH 쉘 환경변수 설정은 디버깅, 개발 시에만 제한적으로 사용되는 것이 바람직합니다.
